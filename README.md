# VLC 拡張: ファイル移動 / ゴミ箱 / ファイル名変更

プレイリスト内の動画ファイルを **別フォルダへ移動** ・ **ゴミ箱へ送る** ・ **ファイル名変更** するための VLC Lua 拡張機能です(Windows 用)。

## できること
- プレイリストのローカルファイル一覧から操作対象を選択(複数選択可)
- 選択ファイルを指定フォルダへ移動
- 選択ファイルをゴミ箱(Recycle Bin)へ移動
- 選択ファイル(1件)を同じフォルダ内でファイル名変更
- 移動先は「お気に入りフォルダのボタン」または「参照...」での任意フォルダ選択の両対応
- 処理後はプレイリストから自動的に項目を削除

## 重要な制約
VLC の Lua 拡張は **Qt プレイリストの右クリックメニューに項目を追加できず、選択状態も取得できません**。
そのため本拡張は「表示」メニューから開くダイアログとして動作し、操作対象は **ダイアログ内の一覧** で選択します。
(真の右クリックメニュー化には VLC 本体を C で改造・ビルドする必要があります。)

## インストール
`filemovedelete.lua` を次のいずれかに配置します。

- ユーザー単位（推奨）: `%APPDATA%\vlc\lua\extensions\`
- 全体: `<VLCインストール先>\lua\extensions\`

フォルダが無ければ作成してください。配置後 VLC を再起動します。

## 使い方
1. メニュー **表示(View) → 「ファイル移動 / ゴミ箱」** を選択(初回は VLC 再起動後）。
2. 一覧から操作対象を選択。
3. 移動の場合:「参照...」かお気に入りボタンで移動先を指定 →「フォルダへ移動」。
4. ゴミ箱の場合:「ゴミ箱へ移動」。
5. 名前変更の場合:**1件だけ選択**し「選択名を読込」で現在名を取り込み、編集して「名前変更」。
   同じフォルダ内でリネームし、プレイリストの項目も新しいパスに差し替わります。

### お気に入りフォルダ
- 移動先パスを入れて「お気に入りに追加」を押すとボタンとして登録されます。
- 登録先ファイル: `%APPDATA%\vlc\lua\extensions\filemovedelete_favorites.txt`（1 行 1 パス。直接編集も可）。

## 仕組み・注意
- 移動とゴミ箱処理は内部で **PowerShell** を非表示ウィンドウで呼び出します
  （移動: `Move-Item`、ゴミ箱: `Microsoft.VisualBasic.FileIO.FileSystem.DeleteFile`）。
- 移動先に同名ファイルが既にある場合は上書きせずスキップします。
- 再生中のファイルはロックされ移動/削除に失敗することがあります。必要に応じて再生を停止してください。
- 動作ログは VLC のメッセージ（ツール → メッセージ、詳細度 2）で確認できます。

## 動作環境
- Windows + VLC（Lua 拡張対応版、VLC 2.x 以降）
